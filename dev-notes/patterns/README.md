## Usage

These files contain detailed implementation examples referenced from `.github/copilot-instructions.md` and summarised in `CLAUDE.md`.

**When to consult:**
- Implementing a new feature using one of these patterns
- Need detailed examples beyond the core guidelines
- Understanding best practices for specific WordPress APIs

**Note:** These are portable reference patterns, not project-specific implementations. For Easy G Maps architecture and conventions, see [`../../CLAUDE.md`](../../CLAUDE.md) and [`../00-project-tracker.md`](../00-project-tracker.md).

> Patterns that did not apply to this plugin (WooCommerce/HPOS, custom database tables, hash-based admin tabs, and the caching/transients pattern) were removed during initial setup. Reinstate from the portable base if a future feature needs them.
